Use npm install to install dependencies.

Pre-Requisites
1. Install and setup mongodb with default settings
2. Setup mongodb server as windows service and start it. This is becuase you will not have to run db service before run application in future.
3. Install latest nodejs.

Install Server
1. Open command prompt and go to root folder where the repository is cloned.
2. run npm install

Install client packages
1. open command prompt and got to angular-src folder
2. run npm install
